Apa itu Coding?

Coding adalah proses menulis instruksi atau perintah yang akan dijalankan oleh komputer. Melalui coding, kita memberikan serangkaian perintah yang membentuk suatu program atau aplikasi, sehingga komputer dapat memahami dan melaksanakan tugas sesuai dengan yang kita inginkan.
Coding juga merupakan keterampilan yang memungkinkan seseorang membuat berbagai perangkat lunak, mulai dari aplikasi, situs web, hingga program komputer.

HTML

HTML, singkatan dari HyperText Markup Language, adalah bahasa standar yang digunakan untuk membuat dan menampilkan halaman web. HTML bukan bahasa pemrograman, melainkan bahasa markup yang menggunakan tag untuk menandai elemen-elemen pada halaman web, seperti teks, gambar, judul, dan tautan.

Struktur Dasar HTML

    Struktur dasar HTML terdiri dari deklarasi seperti:
  • <!DOCTYPE html>
  • <html>
  • <head>
  • <body>

Deklarasi <!DOCTYPE html> memberitahu browser jenis dokumen yang sedang diproses. Elemen <html> adalah elemen akar yang membungkus seluruh halaman web. Elemen <head> berisi informasi meta data seperti judul halaman, dan elemen <body> berisi konten yang ditampilkan di halaman web.

berikut contoh struktur dasar HTML:

                
<!DOCTYPE html>
<html>
<head>
    <title>Contoh Kode</title>
</head>
<body>
    <h1>Judul</h1>
    <p>Paragraf</p>
</body>
</html>

Indentasi

Indentasi adalah praktik menambahkan spasi di awal baris kode untuk membuat struktur penulisan kode lebih jelas dan mudah dibaca. Meskipun tidak mempengaruhi tampilan halaman web secara langsung, indentasi yang baik sangat penting untuk keterbacaan kode agar memudahkan developer untuk melihat hubungan antar elemen.

berikut contoh indentasi yang benar dan salah:

                
//--------BENAR-------- (Rapih dan mudah dibaca)
<!DOCTYPE html>
<html>
<head>
    <title>Contoh Kode</title>
</head>
</html>

//--------SALAH-------- (Tidak rapih dan sulit dibaca)
    <!DOCTYPE html>
<html>
    <head>
                <title>Contoh Kode</title>
</head>
</html>

Elemen HTML

Elemen HTML adalah blok penyusun dasar dalam bahasa markup HTML yang digunakan untuk membuat struktur dan konten halaman web. Elemen HTML memberi tahu browser bagaimana menampilkan dan menafsirkan bagian tertentu dari dokumen. Secara sederhana, elemen HTML terdiri dari tag pembuka, tag penutup, dan konten di antaranya (meskipun beberapa elemen mungkin tidak memiliki tag penutup).
Contoh elemen dasar HTML:

  • div (untuk mengelompokkan elemen)
  • h1 (untuk membuat judul)
  • h2 (untuk membuat subjudul)
  • p (untuk membuat paragraf)
  • a (untuk memasukkan link)
  • img (untuk menampilkan gambar)
  • ul & li (untuk membuat daftar tidak berurutan)
  • ol & li (untuk membuat daftar berurutan)

berikut contoh penggunaan Class dan Id :

                
<!DOCTYPE html>
<html>
<body>
    <div class="halaman-utama" id="halaman-utama">
        <h1 class="judul-gyat" id="judul">Skibidi Toilet</h1>
        <h2 class="subjudul">The legend of brainrot</h2>
        <p class="paragraf">Kamu sigma banget <br>no cap!</p>
        <img src="kucing_uiia.png" class="gambar" >
        <ol class="sipaling-sigma" id="skill">
            Skill:
            <li>Ngoding</li>
            <li>Bisa Masak Indomie</li>
            <li>Teknisi Sound Horeg</li>
        </ol>
    </div>
</body>
</html>

Class & Id

Dalam HTML, class dan id adalah atribut yang digunakan untuk memberikan nama pada elemen, namun memiliki perbedaan utama.
Berikut penjelasan lebih detail:

    Id:
  • Bersifat unik: Setiap elemen HTML hanya boleh memiliki satu id yang unik di dalam satu halaman.
  • Digunakan untuk menargetkan elemen spesifik: styling atau berinteraksi dengan elemen tunggal.
  • Pemanggilan di CSS: #nama-id.


    class:
  • Tidak unik: Satu elemen bisa memiliki beberapa class, dan satu class bisa digunakan di banyak elemen.
  • Digunakan untuk mengelompokkan elemen: styling CSS atau atribut yang sama pada beberapa elemen dengan Class yang sama.
  • Pemanggilan di CSS: .nama-class.

CSS

CSS atau Cascading Style Sheets adalah bahasa yang digunakan untuk mengatur tampilan dan tata letak halaman web. Dengan CSS, kita dapat mengubah warna, ukuran font, margin, padding, dan banyak aspek visual lainnya dari elemen HTML. CSS memungkinkan kita untuk memisahkan konten HTML dari style tampilan, sehingga membuat pengembangan web lebih terstruktur dan mudah dikelola.

Penulisan CSS

Penulisan CSS (Cascading Style Sheets) dapat dilakukan dengan tiga cara utama: inline style, internal style, dan external style.
Berikut adalah penjelasan lebih detail:

    Inline Style:
  • CSS ditulis langsung di dalam tag HTML menggunakan atribut style.
  • Contoh:
    •                         
<html>
<body>
    <p style="color: blue;">Teks ini berwarna biru</p>.
</body>
</html>
  • Cara ini cocok untuk gaya yang spesifik untuk satu elemen saja, tetapi kurang efisien untuk banyak elemen

    • Internal Style:
  • CSS ditulis di dalam tag <style> yang ditempatkan di bagian <head> dari dokumen HTML.
  • Contoh:
    •                         
<head>
    <style>
        //memanggil elemen p
        p { 
            color: blue;
        }

        //memanggil class paragraf
        .paragraf {
            font-size: 16px;
        }

        //memanggil id judul
        #judul {
            font-weight: bold;
        }
    </style>
</head>
  • Cara ini lebih baik daripada inline style untuk gaya yang diterapkan pada beberapa elemen di halaman yang sama

    • External Style:
  • Penulisan dilakukan pada file terpisah dengan ekstensi .css.
  • File CSS ini kemudian dihubungkan ke file HTML menggunakan tag <link> di bagian <head>.
  • Contoh:
    •    konfigurasi file index.html 
                            
//menghubungkan file CSS ke HTML
<head>
    <link rel="stylesheet" href="style.css">
</head>
       konfigurasi file style.css 
                            
//memanggil elemen p
p { 
    color: blue;
}

//memanggil class paragraf
.paragraf {
    font-size: 16px;
}

//memanggil id judul
#judul {
    font-weight: bold;
}
  • Cara ini adalah yang paling umum dan direkomendasikan untuk proyek web yang lebih besar karena memungkinkan pemisahan kode HTML dan CSS, serta memudahkan pengelolaan dan styling.

Properti CSS

Properti CSS adalah aturan yang digunakan untuk menentukan tampilan elemen HTML pada halaman web. Properti ini memungkinkan Anda untuk mengatur berbagai aspek visual, seperti warna, ukuran, font, tata letak, dan banyak lagi.


    Beberapa contoh properti CSS yang umum digunakan meliputi:
  • color: Mengatur warna teks.
  • font-size: Mengatur ukuran teks.
  • background-color: Mengatur warna latar belakang.
  • margin: Mengatur margin (ruang kosong di sekitar elemen).
  • padding: Mengatur padding (ruang kosong di dalam elemen).
  • width: Mengatur lebar elemen.
  • height: Mengatur tinggi elemen.
  • position: Mengatur posisi elemen.
  • border: Mengatur garis tepi elemen.

berikut contoh penggunaan properti CSS:

                
p {
    /* Mengatur warna teks menjadi biru */
    color: blue;

    /* Mengatur ukuran teks menjadi 16 piksel */
    font-size: 16px;

    /* Mengatur warna latar belakang menjadi kuning */
    background-color: yellow;

    /* Mengatur margin elemen menjadi 10 piksel */
    margin: 10px;

    /* Mengatur padding elemen menjadi 5 piksel */
    padding: 5px; 

    /* Mengatur lebar elemen menjadi 300 piksel */
    width: 300px; 

    /* Mengatur tinggi elemen menjadi 100 piksel */
    height: 100px; 

    /* 
    Mengatur posisi elemen 
    relatif terhadap posisinya sendiri 
    */
    position: relative;

    /* 
    Mengatur garis tepi elemen dengan lebar 1 
    piksel, solid, dan warna hitam 
    */
    border: 1px solid black; 
}
